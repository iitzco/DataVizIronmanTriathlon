Description

This visualization shows and compares total race time as well as each discipline race time.
Each athlete is represented by a line where its length is proportional to the time . The chart can be filtered by genre, country or category. Each athlete is shown with the time spent on the corresponding discipline and with total and relative overall ranking.
This visualization represents that triathlon is a sport where you need to be consistent in all disciplines in order to be succesfull. Triathlon champions are not the best swimmers, cyclists or runners.
This visualization was implemented using Tableu.
